# MyGin

## Gin介绍

- Gin是一个Golang的框架

## Gin安装

官方文档

```
https://gin-gonic.com/docs/quickstart/
```

源码

```
https://github.com/gin-gonic/gin
```

下载安装

```
go get -u -v github.com/gin-gonic/gin
```

下载之前需要设置代理

开启Go Module

```
go env -w GO111MODULE=on
```

开启代理

```
go env -w GOPROXY=https://goproxy.cn,direct
```

查看环境变量修改

```
go env
```

初始化模块

```
go mod init xuqiulin.com/mygin
```
